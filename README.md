# DevOps CI/CD Project

This project demonstrates a simple CI/CD pipeline using GitHub Pages.

## Project Overview
A static HTML website is automatically deployed whenever changes are pushed to the main branch.

## Tools Used
- GitHub
- GitHub Pages
- GitHub Actions
- HTML

## Features
- Automated deployment
- Version control using Git
- Live hosted website
